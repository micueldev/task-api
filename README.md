## Running the app
1. Clone the project
2. Install dependencies
```bash
$ npm install
```
3. Clone the file ```.env.template``` and rename it to ```.env```, edit the new file if necessary
4. Run the app
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
